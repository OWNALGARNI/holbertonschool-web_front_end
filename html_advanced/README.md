## Description
The goal of this project is to learn how to structure a web page using **HTML5** only.
No CSS or styling is used. The focus is on semantic HTML and correct page structure.

## Learning Objectives
- Understand the structure of an HTML5 page
- Use semantic HTML tags correctly
- Organize content using header, main, footer, section, article, and aside
- Use headings, lists, tables, links, and media elements properly
- Validate HTML structure and follow best practices
